# 🎬 Entertainment Popularity & Rating Prediction

## 📌 Project Overview

This project applies machine learning techniques to predict and analyze the popularity and ratings of TV series/anime using real-world datasets.

We use both **Supervised** and **Unsupervised Learning** approaches:

- Predict ratings (Regression)
- Classify shows (Hit vs Standard)
- Cluster shows into audience segments
